Co-Traveller

Ο Co-Traveller κάνει το δύσκολο κομμάτι του ταξιδιού… απλό.

Περιγραφή Εφαρμογής

Ο Co-Traveller είναι μια ολοκληρωμένη mobile εφαρμογή που λειτουργεί ως intelligent ταξιδιωτικός σύμβουλος και αναλαμβάνει να σου λύσει τα παρακάτω προβλήματα:

Ταξιδιωτικά έγγραφα
Ώρα άφιξης πτήσης & live tracking
Ισοτιμία
Εκτός σύνδεσης πλοήγηση
Καιρός
Προσωποποιημένος ταξιδιωτικός οδηγός με τεχνητη νοημοσύνη

UI Process

Step 1

Εισάγεται κωδικό κράτησης πτήσης & όνομα Ξενοδοχείου

Step 2

Έχετε έτοιμη και προσωποποιημένη εμπειρία για τον προορισμό και την πτήση σας

⚡ Βασικές Λειτουργίες

🛫 Flight Tracking

Real‑time παρακολούθηση πτήσεων

📋 Tαξιδιωτικά Έγγρφα

Αυτόματος έλεγχος visa/passport, και υπενθύμιση την ημέρα πτήσης.

🗺️ Πλοήγηση Από και Πρός το Αεροδρόμιο

Βέλτιστες διαδρομές αεροδρόμιο ↔ ξενοδοχείο με offline οδηγίες.

📱 Must visit LLM
Tαξιδιωτικός οδηγός

Προσωποποιημένες προτάσεις για αξιοθέατα & φαγητό με LLM.

💱 Αντιστοιχία Νομισμάτων

Live ισοτιμίες και γρήγορες μετατροπές.

🧳 e‑memories passport

Προσωπικό αρχείο με προορισμούς, ξενοδοχεία και φωτογραφίες.

Χρονοδιάγραμμα

Φάση 1 (1-2 Εβδομάδες)

Έρευνα αναγκών χρηστών, Research APIs, UI design, mobile development setup

Φάση 2 (3 Μήνες)

React native expo FE & node BE build, include APIs, database integration, LLM integration

Φάση 3 (1-2 Μήνες)

Testing, optimization, documentation

🎯 Μαθησιακοί Στόχοι

  • API integration & Data-driven UX: αξιοποίηση επίσημων APIs και προσωποποιημένων εμπειριών χρήστη
  • Ενσωμάτωση LLM : χρήση γλωσσικών μοντέλων για προσωποποιημένο περιεχόμενο, βελτίωση UX και αξιοποίηση AI capabilities
  • Full Stack Mobile Development: ανάπτυξη με React Native, Node.js & SQLite
  • Real-Time Εμπειρία Χρήστη: offline/online λειτουργίες, ειδοποιήσεις, χάρτες, real-time δεδομένα
  • Πλήρες Testing: δυναμική προσαρμογή της εμπειρίας χρήστη βάσει εισροών
  • Αξιολογόγηση χρηστών & Documentation: έλεγχοι λειτουργικότητας, usability testing, τεχνική τεκμηρίωση
