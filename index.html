Co-Traveller

Ο Co-Traveller κάνει το δύσκολο κομμάτι του ταξιδιού… απλό.

Περιγραφή Εφαρμογής

O Co-Traveller είναι μια ολοκληρωμένη mobile εφαρμογή που λειτουργεί ως intelligent ταξιδιωτικός σύμβουλος και αναλαμβάνει να σου λύσει τα παρακάτω προβλήματα:

  • Ταξιδιωτικά έγγραφα
  • Ώρα άφιξης πτήσης & live tracking
  • Ισοτιμία
  • Offline Navigation
  • Καιρός
  • Personalised προτάσεις

UI Process

Step 1

Εισάγεται κωδικό κράτησης πτήσης & όνομα Ξενοδοχείου

Step 2

Έχετε έτοιμη και προσωποποιημένη εμπειρία για τον προορισμό και την πτήση σας

⚡ Βασικές Λειτουργίες

🛫 Flight Tracking

Real-time παρακολούθηση πτήσεων, automatic scheduling για την ώρα άφιξης στο αεροδρόμιο notifications για καθυστερήσεις
πηγή: AviationStack

📋 Tαξιδιωτικά Έγγρφα

Automated visa/passport checking, Ενημέρωση για τα έγγραφα που χρειάζονται κατά την κράτηση και υπενθύμιση τη μέρα της πτήσης. Στοιχεία απο επίσημες πηγές.

+ Αντιστοιχία νομισμάτων

🗺️ Πλοήγηση Από και Πρός το Αεροδρόμιο

Optimized routes από αεροδρόμιο σε ξενοδοχείο, offline οδηγίες σε local storage

📱 Live LLM
Tαξιδιωτικός οδηγός

Προτάσεις για must-see με χρήση LLM μέσω του OpenAI API για προσωποποιημένες συστάσεις.
+ όμορφο αρχείο ταξιδιών του χρήστη ως ενθύμιο "e-memories passport" με τους προορισμούς, το ξενοδοχείο και τα έξοδα.

Χρονοδιάγραμμα

🎯 Μαθησιακοί Στόχοι

  • API integration & Data-driven UX: αξιοποίηση επίσημων APIs και προσωποποιημένων εμπειριών χρήστη
  • Ενσωμάτωση LLM (OpenAI API): χρήση γλωσσικών μοντέλων για προσωποποιημένο περιεχόμενο, βελτίωση UX και αξιοποίηση AI capabilities
  • Πολυτροπική Αλληλεπίδραση: offline/online λειτουργίες, ειδοποιήσεις, χάρτες, real-time δεδομένα
  • Full Stack Mobile Development: ανάπτυξη με React Native ή Flutter, Node.js & SQLite
  • Feedback Loops: δυναμική προσαρμογή της εμπειρίας χρήστη βάσει εισροών
  • Πλήρες Testing & Documentation: έλεγχοι λειτουργικότητας, usability testing, τεχνική τεκμηρίωση
